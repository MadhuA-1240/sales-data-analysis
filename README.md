# Sales Data Analysis

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on sales data to identify trends, analyze regional performance, and generate business insights.

## 🎯 Objectives
- Analyze total and average sales
- Compare sales performance across regions
- Identify high-performing segments
- Visualize sales distribution

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib

## ⚙️ Features
- Data preprocessing and cleaning
- Aggregation of sales by region
- Statistical analysis (mean, total)
- Visualization using bar charts

## 📊 Results
- Identified region-wise sales performance
- Calculated total and average sales
- Visualized sales trends for better decision-making

## 🚀 Future Improvements
- Use real-world datasets
- Add interactive dashboards (Power BI)
- Apply predictive analytics
